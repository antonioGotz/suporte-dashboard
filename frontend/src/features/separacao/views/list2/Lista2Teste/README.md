Lista2Teste

Esqueleto para a nova lista de separação (Lista2Teste). Esta pasta contém:

- Lista2Teste.jsx (componente gerente)
- Lista2TesteItem.jsx (subcomponente para item - card/row)
- Lista2Teste.styles.js (styled tokens e componentes básicos)
- Lista2Teste.utils.js (funções utilitárias)
- Lista2Teste.hooks.js (hook para ações comuns)

Status: esqueleto criado. Ainda não integrado ao `SeparacaoPage.jsx`. Para integrar, passar `onSetStatus`, `onGenerateLabel`, `statusPipeline`, `stageByCode`, `updatingIds` como props.
